# Trust Demo App

## Installation
En el virtual environment que esta instalado trust-monitor:

```sh
pip install streamlit
```

## Usage
Ir hasta el directorio de la demo y correr el run.

```sh
cd ...\trust-demo 
streamlit run Home.py
```
